# Features

## User Types

| User | Description |
|------|-------------|
| **Artist/Band** | Musicians coordinating rehearsals, setlists, gigs |
| **Venue** | Businesses booking acts, managing shows |
| **Fan** | Music lovers discovering local live music |

## MVP (Phase 1) — Band Tools

Target user: Artist/Band

- **Setlists** — Create, edit, share setlists for gigs
- **Rehearsal scheduling** — Coordinate practice times with bandmates
- **Gig calendar** — Track upcoming shows

Core action: *Create and share a setlist for an upcoming gig.*

## Phase 2 — Show Discovery

Target user: Fan

- **Event listings** — Browse local live music
- **Filters** — By genre, date, location
- **Facebook sync** — Auto-import events artists post elsewhere

## Phase 3 — Booking Marketplace

Target users: Artist/Band, Venue

- **Artist profiles** — Genre, media, availability
- **Venue profiles** — Capacity, typical genres, pay structure
- **Booking flow** — Request → negotiate → confirm
- **Payments** — Escrow, payouts (monetization opportunity)
